# 达观数据项目实施

## VOC-AI分析系统实施流程

### 阶段一：数据准备
1. 收集 VOC 品类对应的 ID
2. 收集 VOC 品牌对应的 ID
3. 根据品类对应的标签体系（先提供砂锅作为示例）
4. 分析样例

### 阶段二：AI分析实施
**系统处理逻辑**：
```
从VOC系统爬取电商评论原文
    → AI打上自定义的各品类标签
    → 客户声音分析
    → 标签结果导入PowerBI
    → 业务用户自定义各维度分析
```

### 阶段三：业务应用
- **图表用途**：业务对公司产品的质量分析、竞品分析等报告
- **目标受众**：业务分析师、管理层
