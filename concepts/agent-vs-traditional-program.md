# Agent vs 传统程序

## 核心区别

| 维度 | 传统程序 | AI Agent |
|------|---------|----------|
| 交互方式 | 固定指令流程 | 目标驱动、自主规划 |
| 输出 | 生成文本 | **生成行动并执行行动** |
| 适应性 | 需手动调整 | 自我调整策略 |
| 比喻 | 自动售货机：投币→按按钮→出商品 | 私人助理：告诉需求→助理规划→完成任务 |

## 思维转变

> 从对话框问答进化为**目标驱动的任务执行**
