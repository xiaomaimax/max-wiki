# ERP生产模块

## 概述
ERP生产模块覆盖从生产计划到数据收集的全流程，核心挑战在于设计与自身企业匹配的模块，并保证数据的准确性。

## 核心模块
- **MPS**（Master Production Schedule）：生产主计划
- **MRP**（Material Requirements Planning）：物料需求计划，MRP = MPS - 库存
- **PR**（Purchase Request）：采购申请，PR = MRP - 库存

## 关键议题
1. 如何设计与自身企业相匹配的 ERP 生产模块
2. 如何在 ERP 生产模块中及时变更 / 变动信息
3. 如何保证 ERP 生产模块中数据的正确性
4. 如何利用生产软件进行生产信息收集
5. 如何保证生产管理软件所收集信息的准确性
6. 如何进行生产报表更新管理
