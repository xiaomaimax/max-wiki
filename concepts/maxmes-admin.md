# MaxMES 系统管理员指南

## 系统配置

### 基础数据配置
- 工厂日历设置
- 班次时间配置
- 工序定义
- 工艺路线
- BOM清单维护

### 系统参数
- MES系统参数配置
- 审批流程配置
- 消息通知配置

## 用户权限管理

### 用户创建
1. 输入工号、姓名、部门
2. 设置登录账号和密码
3. 分配角色

### 角色权限矩阵

| 角色 | 权限范围 |
|------|----------|
| 系统管理员 | 全部权限 |
| 生产主管 | 生产模块全部权限 |
| 车间主任 | 车间数据查看、报工 |
| 操作工 | 报工、点检 |
| 仓库管理员 | 仓库模块权限 |

## 系统维护

### 数据备份
- 数据库全量备份（每日）
- 配置文件备份（变更时）
- 备份文件保留30天

### 日志管理
- **操作日志**：记录用户关键操作
- **系统日志**：记录系统异常
- **接口日志**：记录与外围系统交互

### 系统监控
- CPU/内存/磁盘监控
- 数据库连接池监控
- 接口响应时间监控
